# Responsive Portfolio WebPage

## Overview
A fully responsive personal portfolio showcasing my skills, projects, and resume. Built with HTML, CSS, and JavaScript.

## Features
- Responsive layout across devices
- Sections: About, Projects, Skills, Resume, Contact
- Recruiter-friendly presentation

## Tech Stack
HTML, CSS, JavaScript, Bootstrap 5

## How to Run
1. Clone or download the repo
2. Open `web.html` in your browser
